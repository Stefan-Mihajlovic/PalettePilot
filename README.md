# Palette Pilot

Palette Pilot is a local-first browser color workspace. Version 2 keeps the fast screen picker and adds saved palettes, creation tools, extraction, accessibility analysis, semantic tokens, side-panel mode, and production-ready exports.

## Free

- Unlimited color picking
- HEX, RGB, HSL, and OKLCH values
- Six recent colors
- Quick copy
- One saved palette
- Basic contrast checking
- TXT and CSS export

## Pro — $9.99 lifetime

- Unlimited named palettes and folders
- Page and image palette extraction
- Shade, tint, harmony, gradient, and light/dark theme generation
- Complete accessibility studio and color-blindness simulation
- Semantic color tokens
- Side-panel workspace
- CSV, CSS, SCSS, Tailwind, JSON, SVG, PNG, ASE, and PDF exports
- Local backup and import
- Up to three active installations
- All future Pro updates

Palette data stays in local extension storage. Image analysis happens locally. Page extraction runs only after the user invokes Palette Pilot on the active page.
